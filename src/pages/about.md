Rem1L 是一名 adversarial LLM engineer，研究方向集中在 LLM 安全领域：

- **Indirect Prompt Injection (IPI)** — 针对 LLM agent 的间接提示注入攻击与防御
- **Agent 系统的 Access Control** — 多 agent 架构下的权限边界与隔离机制
- **传统防御方法的有效性** — 经典安全防御在面对 LLM-assisted 攻击时的局限性分析
- **LLM-assisted 漏洞发掘** — 利用 LLM 能力探索新型漏洞发现技术

目前在 Ohio State University (OSU) 攻读 PhD 学位，持续关注 LLM 安全前沿，积极寻找有趣的合作 project。

如有合作意向或想法，欢迎通过 GitHub 联系。
