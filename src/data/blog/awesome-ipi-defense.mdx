<En>
I was trying to find a good list of IPI defense papers. [`tldrsec/prompt-injection-defenses`](https://github.com/tldrsec/prompt-injection-defenses) came up — last updated over a year ago. Dozens of arxiv papers nowhere in sight.

So I had cc make one: [`Rem1L/awesome-ipi-defense`](https://github.com/Rem1L/awesome-ipi-defense).

To keep it from rotting the same way, there's a weekly GitHub Action that queries Semantic Scholar for papers citing the existing seeds and opens an issue with candidates for manual review. It's citation-based, so papers that don't cite anything already in the list can still slip through — but it catches most of the incremental work automatically.

PRs welcome for the gaps.
</En>
